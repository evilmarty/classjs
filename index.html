class.js by Marty Zalega

A nifty utility to craft javascript objects as a sir.

Creating a class is as simple as sipping on a glass of whisky:

var Gentleman = Class({
  initialize: function() {
    alert("splendid!");
  }
})

Then using your newly crafted class is just like taking another sip:

var sir = new Gentleman()

Sub-classing is the same as tipping your hat:

var ClassyGentleman = Gentleman({
  yes: function() {
    alert('quite');
  }
})
